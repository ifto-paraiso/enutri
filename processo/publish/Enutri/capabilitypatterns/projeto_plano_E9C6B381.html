Main Description

Este artefato define os parâmetros para o acompanhamento do progresso do projeto e especifica os objetivos de alto nível das iterações e seus marcos. Descreve também como o projeto está organizado e quais papéis são executados por quem.

O gerente de projeto é responsável por desenvolver o plano de projeto, trabalhando em conjunto com o resto da equipe. O plano de projeto permite que os Stakeholders e os outros membros da equipe tenham uma panorâmica do projeto e uma idéia de que quando esperar um determinado nível de funcionalidade. Ele também é usado para registrar as lições aprendidas no projeto.
