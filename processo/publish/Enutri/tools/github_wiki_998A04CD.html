Main Description

O GitHub Wiki é um lugar no repositório GitHub onde é possível compartilhar conteúdos de longa duração sobre de um projeto, por exemplo: a forma de usá-lo, como ele foi projetado, seus princípios fundamentais, e assim por diante.

Com as páginas wiki torna-se possível escrever o conteúdo assim como em qualquer outro lugar no GitHub. O GitHub usa o código-fonte aberto biblioteca Markup para converter formatos diferentes em HTML, então você pode optar por escrever em Markdown, RST ou qualquer outro formato suportado quando você páginas oficiais wiki.

Wikis podem ser editados diretamente no GitHub, ou você pode trabalhar com um editor de texto offline e simplesmente "commitar" as alterações. Wikis são fundamentalmente colaborativas. Por padrão, apenas colaboradores de seu repositório podem fazer alterações nas wikis, mas você pode configurar isso para ser habilitado para todos os usuários em repositórios públicos.
